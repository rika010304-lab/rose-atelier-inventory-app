# 🌹 Rose Atelier - バラ注文・在庫管理アプリ

元花市場での実務経験をもとに制作した、バラ専門の在庫・注文・廃棄管理アプリです。
現場で発生しやすい在庫ロスや入力ミスの課題を解決することを目的に開発しました。

---

## 📌 概要

花屋業務では、在庫・注文・廃棄の管理が分断されていることにより、
在庫ロスや入力ミスが発生しやすいという課題があります。

本アプリではそれらを一元管理し、
「在庫の見える化」と「ロスの削減」を実現することを目的としています。

## ✨ 主な機能

### 在庫管理

* 品種・色・価格・在庫数の登録
* 在庫一覧表示
* 売り切れ・残り少なめの自動表示

### 注文機能

* カート機能
* 在庫数チェック（超過防止）
* 注文時に在庫自動減算

### 廃棄管理

* 廃棄登録
* 在庫自動減算
* 廃棄理由の記録

### データ連携

* 廃棄データをCSV出力
* Python（Streamlit）で可視化

## 概要

花屋業務で発生しやすい在庫管理・注文管理・廃棄記録を、シンプルに管理できるアプリです。

フロントエンド版では、HTML / CSS / JavaScript と localStorage を使って、在庫登録・注文・廃棄による在庫数の変動を実装しています。

また、廃棄データをCSV出力し、Python / Streamlit で可視化できるダッシュボードも作成しました。

- 廃棄データのCSV出力

## 工夫した点

- 元花の仕事の経験を活かし、現場で起こりやすい在庫ロスに着目しました。
- 品種名や色の入力ミスを防ぐため、登録済み在庫から選択できる設計にしました。
- 「日持ち」「香り」など花屋ならではの情報をUIに取り入れました。
- Webデザイン経験を活かし、花屋らしい柔らかい世界観と、管理画面としての見やすさを両立しました。
