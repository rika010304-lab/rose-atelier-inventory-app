# Rose Atelier 🌹

### バラ注文・在庫管理アプリ

Spring Boot / MySQL / Python

花屋での実務経験をもとに制作した、バラ専門の注文・在庫管理Webアプリです。
在庫・注文・廃棄を一元管理し、在庫ロスや入力ミスの削減を目的として設計・実装しました。

デザインだけでなく、UI設計・バックエンド・データ分析まで一貫して制作しています。

---

# 📌 概要

花屋業務では、在庫管理・注文管理・廃棄記録が分断されていることで、

* 在庫数が合わない
* 廃棄理由が追えない
* 注文ミスが発生する

といった課題が起こりやすくなります。

本アプリではそれらを一元管理し、
「在庫の見える化」と「ロス削減」を実現することを目的としています。

# ✨ 主な機能

## 在庫管理

* 品種・価格・在庫数登録
* 在庫一覧表示
* 売り切れ / 残り少なめ自動表示

## 注文機能

* 商品一覧表示
* カート機能
* Checkout機能
* 注文時の在庫自動減算

## 廃棄管理

* 廃棄登録
* 廃棄時の在庫自動減算

## 注文履歴

* 注文履歴保存
* 注文日時・数量・金額管理

## Python分析

* 注文履歴CSV分析
* 品種別売上ランキンググラフ生成

---

# 📊 Python分析

注文履歴データをCSV化し、Pythonで売上分析を実施。

* 品種別売上ランキング
* 売上本数集計
* グラフ可視化

を行っています。

---

# 💡 工夫した点

* 花屋での実務経験をもとに、現場課題を反映
* 入力ミス防止のため、登録済み品種から選択するUIを採用

* 在庫状態をリアルタイムで自動更新
